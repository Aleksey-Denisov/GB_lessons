## Списки

Чтобы добавить не нумерованные списки необходимо добавить символ (*). Пример:
* Элемент1
* Элемент2
* Элемент3

Чтобы добавить нумерованные списки их нужно просто пронумеровать. Пример:
1. Первый пункт
2. Второй пункт
